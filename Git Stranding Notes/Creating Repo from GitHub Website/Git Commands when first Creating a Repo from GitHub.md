### …or create a new repository on the command line
``` bash
echo "# Jabba-The-Hut" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/PanmaTG/Jabba-The-Hut.git
git push -u origin main
```

### …or push an existing repository from the command line
```bash
git remote add origin https://github.com/PanmaTG/Jabba-The-Hut.git
git branch -M main
git push -u origin main
```
